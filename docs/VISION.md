# Vision

Sweetie Bot should feel like a small, stage-ready character with a gentle conversational personality, not just a robot that happens to speak.

The near-term goal is not general AI. The near-term goal is believable, operator-safe character coherence:
- a recognizable persona
- short expressive speech
- authored routines
- soft social attention
- audible replies through the robot itself

The system should also be reusable. If another CERBERUS developer wants only the dialogue plugin or only the audio adapter, they should be able to lift those pieces without dragging in the entire project.
