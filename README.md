# 🐴 Sweetie-Bot (CERBERUS Extension)

> “The dream of a magical, expressive, conversational robot companion… but real.”

Sweetie-Bot is an experimental **character robotics layer** built on CERBERUS.

## ✨ What This Is
- Persona system
- Emotes
- Routines
- LLM dialogue
- Audio integration
- Accessories

## ⚠️ Current Status
Version: 0.0.9 — Scaffold Phase

This is NOT a finished system.

## 🧪 Quick Start
Run API and Web UI locally.

## 💖 Why This Exists
To build a robot with personality.
