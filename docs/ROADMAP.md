# Sweetie Bot Roadmap (Detailed)

## Phase 1 — Foundation (Mostly Complete)
- Plugin system
- Dialogue contract
- Routine system
- Runtime API
- Safety baseline

## Phase 2 — Character Continuity
Focus:
- Memory
- State
- Mood

Components:
- Memory plugins
- CharacterStateManager
- MoodEngine

## Phase 3 — Behavior Layer
Focus:
- Decision making

Components:
- BehaviorDirector
- Attention system
- Routine arbitration

## Phase 4 — Perception + World Awareness
Focus:
- Inputs from environment

Components:
- Perception plugins
- WorldState
- Context awareness

## Phase 5 — Voice + Expression
Focus:
- Making character feel alive

Components:
- TTS plugins
- Audio output
- Speech coordination

## Phase 6 — Integration
Focus:
- External systems

Components:
- CERBERUS adapter
- Hardware integration
- Capability negotiation

## Phase 7 — Operator + Observability
Focus:
- Control + debugging

Components:
- Operator dashboard
- Logs + tracing
- Health monitoring

## Phase 8 — Production Readiness
Focus:
- Stability + usability

Components:
- Packaging
- Deployment
- Plugin ecosystem
- Authoring tools

## Final Goal

A fully modular, safe, expressive AI character framework
that can be reused across:
- robots
- virtual agents
- interactive installations
