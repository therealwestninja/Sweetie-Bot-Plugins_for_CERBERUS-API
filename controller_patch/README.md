# Controller Patch Guidance for Integration Pack v2

This folder describes what the CERBERUS Controller App should expose for the real hardware loop.

## Add or verify these controller features
- plugin endpoint configuration
- live plugin health panel
- autonomy mode + current goal display
- focus target display
- social bonding / best-friend display
- docking / charging state display
- peer / Crusader state display
- manual override controls
- hardware adapter state display

## Important
Because both projects are under shared control, the controller may be updated whenever the Sweetie stack needs new visibility or control surfaces.
