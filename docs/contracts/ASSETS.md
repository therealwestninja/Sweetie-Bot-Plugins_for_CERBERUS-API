# Asset Contracts

## Persona asset
YAML file that defines:
- id
- display_name
- traits
- dialogue_style
- motion_style
- safety_profile

## Emote asset
JSON file that defines:
- id
- tags
- default_duration_ms
- body_profile
- linked_accessories

## Routine asset
YAML file that defines:
- id
- steps
- cancellation_behavior
- required_capabilities
